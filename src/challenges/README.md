# Beginner TypeScript Challenges

Each challenge file contains function stubs with `Not implemented` errors.

Your job: implement them until `npm test` is green.

## Challenge goals

1. **01_primitives.ts**
   - basic function signatures
   - unions (`number | null`)
   - input validation

2. **02_collections.ts**
   - generics
   - `readonly` arrays
   - key-based grouping

3. **03_unions.ts**
   - discriminated unions (`Result<T, E>`)
   - narrowing
   - handling `unknown`

4. **04_async.ts**
   - Promises
   - custom errors
   - retry logic
